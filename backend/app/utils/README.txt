MÓDULO: utils
=============
Contiene funciones auxiliares reutilizables en todo el proyecto.

Archivos:
- dependencies.py   → Dependencias de FastAPI (get_db, get_current_user, verificar rol)
- responses.py      → Formatos estándar de respuesta exitosa y de error
- pagination.py     → Lógica de paginación para listados

Este módulo no contiene lógica de negocio, solo herramientas de soporte.
