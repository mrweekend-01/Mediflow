MÓDULO: models
==============
Contiene los modelos SQLAlchemy que mapean las tablas de la base de datos.

Archivos:
- clinica.py        → Tabla clinicas
- usuario.py        → Tabla usuarios
- especialidad.py   → Tabla especialidades
- medico.py         → Tabla medicos
- horario.py        → Tabla medico_horario
- atencion.py       → Tabla atenciones

Cada modelo hereda de Base definido en app/database.py
